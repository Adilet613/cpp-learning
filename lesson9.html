Урок 9: Массивы

Что такое массивы в C++?

Массивы в C++ — это коллекции однотипных данных, хранящихся в памяти подряд. Массивы позволяют эффективно работать с большими объемами данных и ускоряют выполнение программы, так как доступ к элементам массива происходит за постоянное время.

Массивы в C++ индексируются с нуля, что означает, что первый элемент массива имеет индекс 0, второй — 1 и так далее.

Пример:

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5}; // Массив из 5 элементов

    // Выводим элементы массива
    for (int i = 0; i < 5; i++) {
        cout << "Элемент массива " << i << ": " << arr[i] << endl;
    }

    return 0;
}

В этом примере создается массив из 5 элементов, и с помощью цикла for мы выводим значения каждого элемента массива на экран.
