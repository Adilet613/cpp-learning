Урок 4: Условия и операторы

Что такое условия и операторы в C++?

В C++ операторы условных переходов используются для принятия решений в программе. Эти операторы позволяют проверять различные условия и выполнять код в зависимости от результата проверки.

Основной оператор для проверки условий — это if. Он позволяет выполнить блок кода, если условие истинно.

Пример:

if (x > 10) {
    cout << "x больше 10";
} else {
    cout << "x меньше или равно 10";
}

В этом примере, если значение переменной x больше 10, программа выведет "x больше 10". В противном случае — "x меньше или равно 10".
