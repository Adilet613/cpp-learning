Урок 3: Переменные и типы данных

Что такое переменные в C++?

Переменные в C++ — это именованные области памяти, которые хранят данные. Каждая переменная должна иметь тип, который указывает, какие данные она может хранить.

Пример создания переменной:

int age = 25;

Здесь создается переменная age типа int, которая хранит целое число.
