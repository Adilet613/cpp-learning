16. Стандартная библиотека C++ (STL)

Что такое STL

STL (Standard Template Library) предоставляет набор обобщённых классов и алгоритмов для работы с данными.

Пример использования вектора:

#include 
#include 
using namespace std;

int main() {
    vector v = {1, 2, 3, 4, 5};
    for (int i : v) {
        cout << i << " ";
    }
    return 0;
}

Вектор (vector) — это динамический массив, который автоматически увеличивает свой размер при добавлении новых элементов.
