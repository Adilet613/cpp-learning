Многократный доступ в C++

Многократный доступ связан с ситуациями, когда несколько потоков пытаются одновременно работать с одними и теми же данными. Это может привести к ошибкам, если доступ не синхронизирован. Для этого в C++ используют мьютексы.

Пример многократного доступа с синхронизацией:

#include 
#include 
#include 

using namespace std;

mutex mtx;  // Мьютекс для синхронизации доступа

void print_numbers(int id) {
    mtx.lock();  // Захватываем мьютекс
    cout << "Поток " << id << " выводит число" << endl;
    mtx.unlock();  // Освобождаем мьютекс
}

int main() {
    thread t1(print_numbers, 1);  // Создаем первый поток
    thread t2(print_numbers, 2);  // Создаем второй поток

    t1.join();  // Ожидаем завершения первого потока
    t2.join();  // Ожидаем завершения второго потока

    return 0;
}

Этот пример демонстрирует использование мьютекса (mutex) для синхронизации доступа к ресурсу (в данном случае, выводу на экран), чтобы избежать ошибок в многозадачной среде.
