Урок 10: Строки

Что такое строки в C++?

Строки в C++ — это последовательности символов. Строки в C++ могут быть представлены как массивы символов, а также с помощью стандартного типа данных string, который упрощает работу с текстовыми данными.

Строки можно создавать, изменять и манипулировать ими с помощью различных функций и методов, предоставляемых C++.

Пример:

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "Алиса"; // Создаем строку
    cout << "Привет, " << name << "!" << endl; // Выводим строку на экран
    return 0;
}

В этом примере создается строка name и выводится на экран приветственное сообщение.
