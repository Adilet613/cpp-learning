13. Полиморфизм в C++

Что такое полиморфизм

Полиморфизм позволяет использовать один интерфейс для объектов разных типов, обеспечивая переопределение методов в производных классах.

Пример полиморфизма:

#include 
using namespace std;

class Animal {
public:
    virtual void sound() { cout << "Animal makes sound" << endl; }
};

class Dog : public Animal {
public:
    void sound() override { cout << "Woof!" << endl; }
};

int main() {
    Animal* animal = new Dog();
    animal->sound();  // Вызовет метод sound() класса Dog
    delete animal;
    return 0;
}

Полиморфизм позволяет вызывать переопределённый метод на основе типа объекта, несмотря на то, что указатель имеет тип базового класса.
