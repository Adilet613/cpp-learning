11. Операторы сравнения и логические операторы

Операторы сравнения и логические операторы в C++

Операторы сравнения используются для сравнения значений. Логические операторы позволяют комбинировать условия.

Операторы сравнения:

  • == — равенство
  • != — неравенство
  • > — больше
  • < — меньше
  • >= — больше или равно
  • <= — меньше или равно

Логические операторы:

  • && — логическое "И"
  • || — логическое "ИЛИ"
  • ! — логическое "НЕ"

Пример:

#include 
using namespace std;

int main() {
    int a = 5, b = 10;
    
    if (a < b && b > 0) {
        cout << "a меньше b и b положительное" << endl;
    }

    if (a == 5 || b == 5) {
        cout << "a или b равны 5" << endl;
    }

    if (!(a > 10)) {
        cout << "a не больше 10" << endl;
    }

    return 0;
}

Этот код использует операторы сравнения и логические операторы для выполнения различных условий.
