11. Структуры и классы в C++

Что такое структуры и классы

Структуры и классы в C++ используются для объединения данных. Структуры позволяют группировать переменные разных типов, а классы добавляют функциональность через методы.

Пример структуры:

#include 
using namespace std;

struct Person {
    string name;
    int age;
};

int main() {
    Person person1 = {"John", 25};
    cout << "Name: " << person1.name << ", Age: " << person1.age << endl;
    return 0;
}

Этот код создает структуру для представления человека с полями для имени и возраста.
