Урок 6: Массивы

Что такое массивы в C++?

Массивы позволяют хранить несколько значений одного типа в одной переменной. Например, можно создать массив для хранения 5 целых чисел.

Массивы индексируются, начиная с нуля. Каждый элемент массива доступен через свой индекс.

Пример:

int arr[5] = {1, 2, 3, 4, 5};
cout << arr[0]; // Выведет 1

Этот код создает массив arr из 5 элементов и выводит первый элемент массива, который равен 1.
