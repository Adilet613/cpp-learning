19. Реализация структур данных в C++

Что такое структуры данных

Структуры данных — это способы организации и хранения данных, такие как массивы, списки, стеки и очереди.

Пример реализации стека:

#include 
#include 
using namespace std;

int main() {
    stack s;
    s.push(10);
    s.push(20);
    s.push(30);

    cout << "Top element: " << s.top() << endl;  // Отображает верхний элемент
    s.pop();  // Удаляет верхний элемент

    cout << "Top element after pop: " << s.top() << endl;
    return 0;
}

Этот код показывает, как использовать стек, который работает по принципу LIFO (Last In, First Out).
