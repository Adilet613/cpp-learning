12. Наследование в C++

Что такое наследование в C++

Наследование позволяет создавать новые классы на основе существующих, наследуя их свойства и методы.

Пример наследования:

#include 
using namespace std;

class Animal {
public:
    void eat() { cout << "Eating food" << endl; }
};

class Dog : public Animal {
public:
    void bark() { cout << "Barking!" << endl; }
};

int main() {
    Dog dog;
    dog.eat();  // Наследованный метод
    dog.bark(); // Метод класса Dog
    return 0;
}

Класс Dog наследует от класса Animal метод eat() и добавляет собственный метод bark().
