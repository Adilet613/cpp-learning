8. Функции и многократный доступ в C++

Функции в C++

Функции — это самостоятельные блоки кода, которые можно вызывать по имени. Они могут принимать параметры и возвращать значения. Это позволяет избежать дублирования кода и улучшить читаемость программы.

Пример использования функции:

#include 
using namespace std;

int add(int a, int b) {
    return a + b;  // Функция для сложения двух чисел
}

int main() {
    int result = add(5, 3);  // Вызов функции add
    cout << "Результат сложения: " << result << endl;
    return 0;
}

Здесь мы создали функцию `add`, которая принимает два параметра и возвращает их сумму. В функции `main` мы вызываем `add` с аргументами 5 и 3 и выводим результат.

Многократный доступ

Многократный доступ может быть связан с многозадачностью или многопоточностью, когда несколько потоков пытаются одновременно работать с одними и теми же данными. Это требует синхронизации, чтобы избежать ошибок.

Пример многократного доступа с синхронизацией:

#include 
#include 
#include 

using namespace std;

mutex mtx;  // Мьютекс для синхронизации доступа

void print_numbers(int id) {
    mtx.lock();  // Захватываем мьютекс
    cout << "Поток " << id << " выводит число" << endl;
    mtx.unlock();  // Освобождаем мьютекс
}

int main() {
    thread t1(print_numbers, 1);  // Создаем первый поток
    thread t2(print_numbers, 2);  // Создаем второй поток

    t1.join();  // Ожидаем завершения первого потока
    t2.join();  // Ожидаем завершения второго потока

    return 0;
}

Этот пример демонстрирует использование мьютекса (`mutex`) для синхронизации доступа к ресурсу (в данном случае, выводу на экран), чтобы избежать ошибок в многозадачной среде.
