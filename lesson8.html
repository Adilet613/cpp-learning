8. Функции в C++

Функции в C++

Функции — это самостоятельные блоки кода, которые можно вызывать по имени. Они могут принимать параметры и возвращать значения. Это позволяет избежать дублирования кода и улучшить читаемость программы.

Пример использования функции:

#include 
using namespace std;

int add(int a, int b) {
    return a + b;  // Функция для сложения двух чисел
}

int main() {
    int result = add(5, 3);  // Вызов функции add
    cout << "Результат сложения: " << result << endl;
    return 0;
}

Здесь мы создали функцию add, которая принимает два параметра и возвращает их сумму. В функции main мы вызываем add с аргументами 5 и 3 и выводим результат.
