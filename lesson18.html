18. Многозадачность и потоки в C++

Что такое многозадачность и потоки

Многозадачность позволяет выполнять несколько задач одновременно. Потоки позволяют программе работать с несколькими процессами одновременно.

Пример работы с потоками:

#include 
#include 
using namespace std;

void print_message() {
    cout << "Hello from thread!" << endl;
}

int main() {
    thread t(print_message);  // Создание потока
    t.join();  // Ожидание завершения потока
    return 0;
}

Этот код создает новый поток, который выполняет функцию print_message.
