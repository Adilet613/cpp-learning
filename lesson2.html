Первая программа на C++

Давай напишем первую программу на C++. Она просто выведет текст "Привет, мир!".

#include <iostream>
using namespace std;

int main() {
    cout << "Привет, мир!" << endl;
    return 0;
}

Скопируй этот код в онлайн-компилятор или на свой компьютер и запусти его.
