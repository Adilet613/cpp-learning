7. Исключения и обработка ошибок в C++

Что такое исключения

Исключения — это способ обработки ошибок в программе. Когда в программе возникает ошибка, она может быть "брошена" в виде исключения, которое затем можно "поймать" и обработать.

Пример обработки исключений:

#include 
#include   // Для std::runtime_error

using namespace std;

void divide(int a, int b) {
    if (b == 0) {
        throw runtime_error("Деление на ноль невозможно!");  // Генерация исключения
    }
    cout << "Результат деления: " << a / b << endl;
}

int main() {
    try {
        divide(10, 0);  // Попытка деления на ноль
    } catch (const runtime_error& e) {  // Обработка исключения
        cout << "Ошибка: " << e.what() << endl;
    }
    return 0;
}

Этот пример показывает, как генерировать исключения с помощью throw и обрабатывать их с помощью try-catch.
