14. Шаблоны в C++

Что такое шаблоны в C++

Шаблоны позволяют создавать обобщённый код, который работает с разными типами данных.

Пример шаблона функции:

#include 
using namespace std;

template 
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(3, 4) << endl;      // для int
    cout << add(3.5, 4.5) << endl;  // для double
    return 0;
}

Шаблон функции add работает с любыми типами данных, которые поддерживают операцию сложения.
