Урок 5: Циклы

Что такое циклы в C++?

Циклы позволяют повторить выполнение блока кода несколько раз. В C++ есть несколько типов циклов, включая for, while и do-while.

Самый простой цикл — это цикл for, который используется, когда мы заранее знаем, сколько раз нужно выполнить блок кода.

Пример:

for (int i = 0; i < 5; i++) {
    cout << "Число: " << i << endl;
}

Этот цикл выведет числа от 0 до 4.
