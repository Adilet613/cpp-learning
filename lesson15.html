15. Исключения и обработка ошибок в C++

Что такое исключения

Исключения используются для обработки ошибок, возникающих во время выполнения программы.

Пример обработки исключений:

#include 
using namespace std;

int divide(int a, int b) {
    if (b == 0) {
        throw "Деление на ноль!";
    }
    return a / b;
}

int main() {
    try {
        cout << divide(10, 2) << endl;
        cout << divide(10, 0) << endl;  // Это вызовет исключение
    } catch (const char* msg) {
        cout << "Ошибка: " << msg << endl;
    }
    return 0;
}

Этот код демонстрирует ловлю исключений с помощью конструкции try-catch.
